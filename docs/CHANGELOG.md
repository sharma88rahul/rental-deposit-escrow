# Changelog

All notable changes to this project will be documented in this file.

---

## [1.0.0] - 2026-07-05

### Added
- **Smart Contracts (Phase 2)**: Rental Agreement and Escrow Soroban contracts in Rust.
- **Frontend Architecture (Phase 3)**: Next.js 15, React App Router, custom global navigation, and theme layouts.
- **Wallet Authentication (Phase 4)**: Stellar Wallets Kit support for Freighter, Albedo, and xBull wallet providers.
- **Rental Agreement Module (Phase 5)**: Automated Zod form validations, state tracking, and drafting workflows.
- **Escrow Payout Module (Phase 6)**: Vaulting locks, refund deduction request triggers, and arbitrator settlements.
- **Activity & Transaction Logs (Phase 7)**: Live on-chain event stream monitoring and retry logs.
- **Analytics & Custom SVG Charts (Phase 8)**: Area, Line, Bar, and Donut SVG chart libraries.
- **Integration Test Suite (Phase 9)**: 32/32 tests coverage verifying full state transitions.
- **CI/CD Pipeline (Phase 10)**: Automated GitHub workflows checking Cargo lints and Next.js builds.
- **Soroban Deployer (Phase 11)**: JS deployment script compiling and deploying contracts on Testnet.
