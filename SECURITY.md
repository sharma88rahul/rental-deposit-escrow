# Security Policy

## Supported Versions

The following versions of RentSure are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of RentSure seriously. If you find a security vulnerability, please do NOT create a public issue. Instead, report it directly to the core developers.

To report a vulnerability:
- Send an email to **security@rentsure.io** with details of the vulnerability.
- Include a proof-of-concept, steps to reproduce, or sample code when possible.

You will receive an acknowledgment of your report within 48 hours, along with a timeline for fixing the issue.
